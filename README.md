Collab Hub is a multi-role collaboration platform with separate dashboards for Influencers, Organizers, and Sponsors.  
It is built using HTML, CSS, and JavaScript with a simple and clean file structure.

---


- Multi-role user flow  
- Clean and modular JavaScript file structure  
- Separate dashboards for **Influencer / Organizer / Sponsor**  
- Simple profile page  
- Role-based redirection handled on the client side  
- Match AI logic (basic) using `matchAi.js`
- The project runs fully on the front-end  
- Different pages handle different roles and flows  
- All logic is written in the `JS/` folder  
- Redirection and UI interactions are done through JavaScript  
- No backend or server setup required  

---

Just open and run index.html in live server.


Or run using a local live server (VS Code Live Server recommended).
